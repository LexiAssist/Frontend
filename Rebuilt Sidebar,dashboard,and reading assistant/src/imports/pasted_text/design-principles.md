Clean Design Principles
1. Clear Visual Hierarchy

Dominant green sidebar (#3D6E4E or similar) creates strong left-aligned navigation anchor
White content area provides maximum breathing room and focus
Rounded corners throughout softens the UI and feels approachable

2. Generous White Space

Large padding around content cards
Spacious margins between elements
Minimal density - not trying to cram information

3. Card-Based Layout

Hero card with soft blue background (#E3EEFF area) and hexagon pattern
Upload card with dashed border in sage green
Cards have subtle shadows and clear boundaries

4. Consistent Icon System

Icons paired with text labels in the sidebar
Circular icon badges (like the green search icon in header)
Simple, recognizable iconography

5. Calming Color Palette

Primary green: Professional, educational feel
Soft blue: Non-threatening, learning-focused
Sage/mint green: Gentle accent for secondary actions
Neutral whites/grays: Clean background

6. Illustration-Driven

Friendly character illustrations
Decorative patterns (hexagons) that don't distract
Visual storytelling in hero sections

7. Typography Hierarchy

Bold, large headings
Clear body text with good contrast
Consistent font weights for hierarchy

8. Organized Navigation

Collapsed/expandable menu items (StudyBuddy section)
Grouped tools under clear labels ("TOOLS")
Active state indication (white background on "Text to speech")

This design prioritizes accessibility, approachability, and clarity - perfect for a learning platform.
